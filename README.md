# Система управления спортивными соревнованиями
## Цифровизация протоколов спортивных мероприятий
## Описание проекта

Система для автоматизации проведения спортивных соревнований с заменой бумажных протоколов на цифровые.

## Цели

- **Исключение бумажной работы** при проведении соревнований
- **Мгновенная публикация** результатов после ввода судьёй
- **Централизованное хранение** всей информации о соревнованиях
- **Прозрачность результатов** для участников и тренеров
- **Автоматическое определение** мест участников

## Роли пользователей

### Администратор
- `Регистрация` судей в системе
- `Создание карточек` соревнований, участников и дисциплин
- `Назначение судей` на конкретные дисциплины
- `Публикация итоговых` протоколов для печати

### Судья
- `Просмотр списка` назначенных дисциплин
- `Ввод результатов` участников в систему
- `Подтверждение правильности` введённых данных
